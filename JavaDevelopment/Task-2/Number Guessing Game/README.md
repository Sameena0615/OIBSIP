# Number Guessing Game

## Description
The Number Guessing Game is a Java console application where the computer generates a random number between 1 and 100. The player tries to guess the number with the help of hints such as "Too High" and "Too Low". The game also tracks the number of attempts and allows multiple rounds.

## Features
- Random number generation (1 to 100)
- User input using Scanner
- Displays "Too High", "Too Low", or "Correct"
- Maximum of 7 attempts
- Shows remaining attempts
- Displays the correct number if the user loses
- Play Again option
- Round summary after each game

## Technologies Used
- Java
- Eclipse IDE
- Java Scanner
- Random Class

## How to Run
1. Open the project in Eclipse.
2. Run the `NumberGuessingGame.java` file.
3. Enter your guesses in the console.
4. Continue until you guess the correct number or reach the maximum attempts.
5. Choose whether to play another round.
