# ATM Interface

## Objective
A console-based ATM Interface developed using Java.

## Features
- User Login with User ID and PIN
- Maximum 3 Login Attempts
- Deposit Money
- Withdraw Money
- Transfer Money
- Transaction History
- Balance Check
- Quit Option

## Technologies Used
- Java
- OOP
- ArrayList
- Scanner

## Project Structure

ATMInterface/
│── Main.java
│── ATM.java
│── Account.java
│── Bank.java
│── Transaction.java
